# Domisys 🚚📊

**Domisys** is a powerful management system designed for supermarkets to efficiently handle deliveries, inventory, users, and reports. This app streamlines operations, enabling CRUD functionality for orders and products, stock management, task assignments to delivery personnel, and comprehensive data visualization with interactive charts and tables.

## 🛠 Key Technologies

- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **State Management**: Zustand, React Hook Form
- **Validation**: Zod
- **Notifications**: React Toastify
- **UI**: NextUI, Tabler Icons
- **Animations**: Framer Motion

## ✨ Features

- Full CRUD for orders, products, and inventory.
- Interactive dashboards with detailed analytics and reports.
- Modular and scalable architecture.
- Database integration in progress.
